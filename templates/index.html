Kidney Stone Prediction

About Project

This project revolves around predicting kidney stone presence based on patient attributes in the Kidney Stone Prediction Kaggle competition. The primary focus is to explore diverse classification techniques within the dataset to build a complete end-to-end project in predictive modeling. The learning journey involves crucial steps like data preprocessing, feature selection, model training, rigorous evaluation, and strategies for effective model deployment.

Project Workflow:

  1. local virtual environments: Setting up isolated environments using tools like virtualenv to manage dependencies for the project without affecting other projects.
  2. Adding Project to Github: Uploading project files to a GitHub repository for version control, collaboration, and code management.
  3. Structuring Project: Utilizing libraries like Cookiecutter to establish a standardized and organized project structure, aiding in consistency and scalability.
  4. Developing Project Pipelines: Creating workflows or pipelines that automate processes, such as data preprocessing, model training, evaluation, and deployment.
  5. Data Ingestion: Collecting and importing data from various sources into the project environment for analysis and modeling.
  6. Data Transformation: Cleaning, processing, and transforming raw data into a suitable format for analysis and model training.
  7. Model Trainer: Building, training, and optimizing machine learning models using various algorithms and techniques.
  8. Model Monitoring: Implementing systems or tools to continuously monitor the performance and behavior of deployed models.
  9. DVC and MLFLOW: Using Data Version Control (DVC) for managing and versioning data alongside MLflow to track experiments, parameters, and model metrics.
  10. Deploying Model: Transitioning the trained model from development to production, making it available for predictions or use in real-world applications.

Project learning:

  1. DVC, Mlflow, Cookiecutter, Installing Docker Desktop, docker hud, loading data form MySQL, Creating pipline, Github Action, Runners and Building CICD pipeline, Setting up AWS server, Configuring IAM, ECR, EC2, Installing Lunix OS and Runing Commands etc
